# CS-106B 编程抽象 (Programming Abstractions)

本仓库包含了我在斯坦福大学 **CS-106B: Programming Abstractions** 课程中的编程作业和解决方案。

## 项目内容

- **Assignment 1**: 包含 Only Connect, Playing Fair, Sandpiles, Stack Overflow 等练习。
- **Assignment 2**: Fun with Collections (包括 Rising Tides, Rosetta Stone 等项目)。

## 开发环境
- 语言：C++ (使用 Stanford C++ Library)
- IDE：Qt Creator
